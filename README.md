## Descripción
Programa que tiene como menu:
    1. Insertar vertices
    2. Conectar vertices
    3. Verificar si dos vertices son adyacentes
    4. Mostrar nodos conectados a un vertice
    5. Recorrido en anchura
    6. Recorrido en profundidad
    7. Recorrido Dijkstra
    8. Salir
